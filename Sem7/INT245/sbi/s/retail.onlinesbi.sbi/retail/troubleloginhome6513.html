Trouble Logging In

Please note that support for certain issues will be available only after successful login.

  • Frequently Asked Questions

  • Please see if your questions are answered here before you post a complaint.
  • Please remember your Username and note that your Username cannot be changed in future.

  • Question : I have received my username but not password (or) I have received my password but not username.
  • Answer : Please note that username and password are dispatched separately and will reach you within 1 week of each other.

  • Question : I am not able to log in with the username and password that I received.
  • Answer : If you have successfully logged in once and have changed your username and password, then the user id you have received through courier will be deactivated. You need to use the changed user id to login.
